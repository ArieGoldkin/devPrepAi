# DevPrep AI - Interview Preparation Platform

AI-powered interview preparation platform built with Next.js 15 and Claude AI.

## 🚀 Quick Start

```bash
cd frontend
npm install
npm run dev
```

Visit http://localhost:3000

- **Code Quality**: ESLint, Husky, Prettier
